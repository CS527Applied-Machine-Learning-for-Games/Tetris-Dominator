About the game

7 Blocks

The Tetris game has 7 blocks names I, O, T, S, Z, J and L.

Get High Score

Smart AI Agent

Trained using Q-learning, Genetic Algorithm, Deep Q Network, out agents can get over 1 million points.

Our goal

The goal of our project is to build an agent that is able to play Tetris,
and defeat the highest score achievable by a human player in competitive tournaments, which is 999,999.

Methodologies

We adopted a reinforcement learning approach, used Q-learning, genetic algorithms and deep-Q-network to train the agent and compare the relative effectiveness.
